# Superpowers Adapter

这个 adapter 将角色 prompt 映射到 Superpowers-style 方法：

- planning: writing-plans
- implementation: subagent-driven-development
- testing: test-driven-development
- review: requesting-code-review
- fixes: systematic-debugging
- completion: finishing-a-development-branch

最小集成会把这些方法编码进角色合同。后续 adapter 可以直接导入已安装的 Superpowers skills。
