# Vendored Skills

This directory is populated for `standard` and `full` profiles.

The first integration keeps lightweight local summaries of Superpowers-style
skills so projects can work without a global Superpowers installation. These
project-local skills may be customized.

Recommended mappings:

- `brainstorming` -> Spec Creator
- `writing-plans` -> Plan Creator
- `subagent-driven-development` -> Manager / Task execution
- `requesting-code-review` -> Code Reviewer
- `test-driven-development` -> Test Agent
- `systematic-debugging` -> Fix Agent
- `finishing-a-development-branch` -> Commit Agent
- `agentflow-manager-workflow` -> Manager daily workflow, config setup, project
  split, feature advance, and third-party module intake
